# Project Estimator Backend

A Laravel REST API for managing projects, staff salaries, and office expenses. It calculates total project costs based on staff time and operational expenses.

---

##  Tech Stack

- **Laravel 10+**
- **MySQL**
- **PHP 8+**
- **XAMPP / Apache**
- **Composer**
